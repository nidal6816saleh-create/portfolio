Drive Alert and Warning System

Description

The Driver Alert and Warning System is a project that relies on artificial intelligence and computer vision technologies to monitor driver behavior while driving. The system detects drowsiness, distraction, phone use, and then issues audio and visual alerts to prevent accidents.

Key Features

Multiple Case Detection

Comprehensive monitoring system that detects various dangerous behaviors including drowsiness, phone usage, and head turning simultaneously.

Real-time Monitoring

Continuous live monitoring with instant response capabilities. Processes video feed in real-time with minimal latency for immediate alert generation.

Accurate Case Detection

Advanced AI algorithms achieve 90% detection accuracy with sophisticated pattern recognition and machine learning models.

Case Documentation

Automatically logs all detected events, along with the time zone, GPS location, and case type, into an SQLite database for analysis and reporting.

Technologies Used

  • Python (Basic Programming Language).
  • OpenCV (Video and Image Processing).
  • pygame (Audio Alert).
  • Dlib and Mediapipe (Face and Hand Detection and Tracking).
  • SQLite (Database).
  • Tkinter (Graphical User Interface).
  • Email/Telegram Notifications.

Mechanism of Action

The system operates via a front-facing camera that monitors the driver's face in real time. Computer vision algorithms detect hand or facial features, such as the eyes, mouth, and head orientation. When a dangerous situation is detected, such as closing the eyes, yawning, turning the head, using a phone, the system immediately triggers an audible and visual alert to warn the driver. The event is recorded in a database, along with the time and location, allowing for later visualization in a graphical interface and reports to be generated to monitor performance and analyze the causes of distraction.
